## 用途

用于兼容 Claude Messages 请求格式。

## 请求

- Method: `POST`
- URL: `https://api.hubto.ai/v1/messages`
- Auth: `Authorization: Bearer YOUR_API_KEY`
- Content-Type: `application/json`

## 最小请求体

```json
{
  "model": "claude-opus-4-7",
  "max_tokens": 512,
  "messages": [
    {
      "role": "user",
      "content": "请介绍 HubTo。"
    }
  ]
}
```

## 请求示例

```bash
curl https://api.hubto.ai/v1/messages \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "model": "claude-opus-4-7",
    "max_tokens": 512,
    "messages": [
      {
        "role": "user",
        "content": "请介绍 HubTo。"
      }
    ]
  }'
```

## 返回

成功时返回 Claude Messages 相容結構。

目前可用模型建議以模型廣場顯示為準。
