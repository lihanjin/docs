## 取得 API Key

先建立一把可用的 AllToken API Key，並確認目標模型已開通。

如果你想把電台和日常使用分開，建議單獨使用一把 Key。

## 安裝麥悠電台

麥悠電台支援 iOS 與 Android。

先安裝 App，再進入設定頁完成模型接入。

## 設定 AllToken API

在 App 設定中填入：

- API Key：你的 AllToken 金鑰
- Base URL：`https://alltoken.co/v1`
- Model：選擇你已開通的模型，例如 `glm-4` 或 `deepseek-v3`

儲存後先做一次測試，確認模型服務可以正常回應。

## 設定語音

進入語音設定頁後，你可以分別設定主持人和助理的音色。

通常建議先用預設音色，再依內容風格調整語速、音高與音量。

## 建立電台

按新增按鈕建立新的電台。

你可以直接輸入 RSS，或匯入 OPML 檔案。

如果你已經接了 RSSHub 或 WeWe RSS，也可以把常看的媒體來源一起訂閱進來。

## 生成並收聽

打開新聞清單，選擇你要的內容，然後生成一集節目。

你可以決定內容要更詳細、做摘要，或直接略過。

節目生成完成後，回到電台頁面播放即可。

## 使用建議

- 先用短內容驗證鏈路，再處理長篇資訊
- 把電台和語音分開測試，會更容易排查問題
- 正式使用時，建議為麥悠電台單獨準備一把 API Key
- 如果你常切換模型，優先選擇穩定的通用模型

## 總結

麥悠電台適合把訂閱、AI 生成和語音播報串成一條工作流。

接入 AllToken 後，你可以用統一網關管理模型調用，再專注於內容生產與播報效果。
