## 概述

Claude Code 是面向開發者的命令列 AI 助手。

透過接入 AllTokens，你可以把 Claude Code 的請求統一走 AllTokens 網關，方便集中管理 API Key、模型權限、額度與路由策略。

## 快速設定

```bash
export ANTHROPIC_API_KEY="YOUR_API_KEY"
export ANTHROPIC_BASE_URL="https://alltoken.aatest.online"
export ANTHROPIC_MODEL="claude-3-5-sonnet"
claude
```

## 說明

- 請先確認你的 AllTokens 金鑰已開通 Claude 相關模型。
- 如需切換模型，只需修改 `ANTHROPIC_MODEL`。
- 如需持久化設定，請把這些變數寫入 shell 設定檔。
