## 快速設定

先打開 [ChatHub 瀏覽器擴充套件文件](https://doc.chathub.gg/overview/chrome-extension)，並安裝擴充套件。

自訂機器人只在瀏覽器擴充套件版本可用。

在 ChatHub 中選擇 OpenAI 相容或自訂提供方，並填入：

- API Key：`YOUR_API_KEY`
- Base URL：`https://alltoken.co/v1`
- Model：例如 `glm-4`
