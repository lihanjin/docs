## 概述

Claude Code 是一个面向开发者的命令行 AI 助手。

通过接入 Passto AllTokens，你可以把 Claude Code 的请求统一走 Passto AllTokens 网关，便于集中管理 API Key、模型访问、额度和路由策略。

## 前置条件

- 已安装并可以正常使用 Claude Code
- 已拥有可用的 Passto AllTokens API Key
- 网络环境可以访问 `https://alltoken.aatest.online`

## 方法一：通过环境变量配置

### 步骤 1：配置 API 认证

在终端中设置你的 API Key：

```bash
export ANTHROPIC_API_KEY="YOUR_API_KEY"
```

### 步骤 2：配置网关地址

将 Claude Code 的基础地址指向 Passto AllTokens：

```bash
export ANTHROPIC_BASE_URL="https://alltoken.aatest.online"
```

### 步骤 3：选择模型

如果你希望默认使用某个 Claude 模型，可以继续设置：

```bash
export ANTHROPIC_MODEL="claude-3-5-sonnet"
```

### 步骤 4：启动 Claude Code

完成环境变量配置后，正常启动 Claude Code 即可：

```bash
claude
```

## 方法二：写入 shell 配置

如果你希望每次打开终端都自动生效，可以把这些配置写入 `~/.zshrc` 或 `~/.bashrc`：

```bash
export ANTHROPIC_API_KEY="YOUR_API_KEY"
export ANTHROPIC_BASE_URL="https://alltoken.aatest.online"
export ANTHROPIC_MODEL="claude-3-5-sonnet"
```

保存后重新加载 shell：

```bash
source ~/.zshrc
```

## 使用建议

- 先在 Passto AllTokens 控制台中确认你的令牌已启用 Claude 相关模型
- 如需切换模型，只修改 `ANTHROPIC_MODEL` 即可
- 如果你同时管理多个环境，建议为 Claude Code 单独创建一个 API Key

## 常见问题

### 提示认证失败

请确认 `ANTHROPIC_API_KEY` 使用的是 Passto AllTokens 控制台里生成的有效令牌。

### 提示找不到模型

请确认当前令牌具备该模型的访问权限，并且模型名称填写正确。

### 请求没有走 Passto AllTokens

请确认 `ANTHROPIC_BASE_URL` 已设置为：

```txt
https://alltoken.aatest.online
```
