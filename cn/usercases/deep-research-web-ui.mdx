## 概述

Deep Research Web UI 通常需要一个可配置的大模型接口来驱动研究、总结和问答流程。

使用 AllTokens 后，你可以通过统一的 API Key 和网关地址完成接入。

## 推荐配置

如果该工具支持 OpenAI 兼容方式，建议使用以下参数：

- API Key：`YOUR_API_KEY`
- Base URL：`https://alltoken.aatest.online/v1`
- Model：例如 `glm-4`

## 配置步骤

1. 进入 Deep Research Web UI 的设置或环境变量配置页。
2. 将模型服务地址设置为 `https://alltoken.aatest.online/v1`。
3. 填写 AllTokens API Key。
4. 选择一个可用模型并保存。
5. 发起一次研究任务验证结果。

## 常见问题

### 接口能连通，但没有结果

请先检查当前模型是否具备该任务所需的能力，并确认令牌额度是否充足。

### 返回认证失败

请确认 API Key 来自 AllTokens 控制台，并且仍处于启用状态。
