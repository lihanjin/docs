## 概述

ComfyUI 提示词小助手适合在图像工作流里生成、润色和结构化提示词。

将 HubTo 接入后，你可以为提示词生成场景统一配置模型入口，并根据需要切换更适合创作、总结或改写的模型。

## 适用场景

- 你希望在 ComfyUI 工作流中统一生成提示词
- 你希望把提示词改写、扩写和风格化放到同一套模型服务中
- 你希望通过 HubTo 统一管理图像前处理阶段的模型访问

## 前置条件

- 已安装 ComfyUI 提示词小助手
- 已拥有可用的 HubTo API Key
- 当前插件支持自定义模型服务、OpenAI 兼容接口或 HTTP 调用

## 配置步骤

### 步骤 1：进入插件设置

在插件配置中查找模型设置、Provider 配置、Base URL 或 API Key 入口。

### 步骤 2：填写 HubTo 参数

- Base URL：`https://api.hubto.ai/v1`
- API Key：`YOUR_API_KEY`
- Model：选择你在 HubTo 中已开通的文本生成模型

### 步骤 3：保存并进行测试

例如输入一段简单需求，让插件生成一组提示词，确认能够返回内容。

## 使用建议

- 建议先用短提示词测试，确认输出风格符合你的工作流
- 如果你需要多个风格模板，可以在插件层做提示词模板，在 HubTo 层统一管理模型
- 如果插件支持缓存或预设，建议保留一个稳定可用的默认模型

## 常见问题

### 插件没有自定义 Base URL 入口

可以先检查是否支持高级设置、实验功能或 HTTP 节点接入。

### 输出结果为空

请确认插件正确读取了响应中的文本结果，并检查目标模型是否适合做文本生成。

### 鉴权失败

请确认 API Key 正确，且没有额外空格或换行。
