## 概述

Trancy 是一个面向网页与视频场景的翻译工具，支持在设置中接入自定义模型服务。

接入 AllTokens 后，你可以在 Trancy 中统一使用 API Key、模型和网关地址。

## 下载与安装 Trancy

### 方式一：安装浏览器扩展（推荐）

1. 访问 Trancy 下载页：[https://www.trancy.org/download](https://www.trancy.org/download)
2. 在 `Desktop Extension` 区域选择你的浏览器并下载安装
3. 安装完成后，固定 Trancy 扩展图标到浏览器工具栏
4. 点击扩展图标，确认可以正常打开 Trancy 面板

当前下载页提供的主流浏览器包括：

- Chrome
- Edge
- Firefox
- Brave
- Arc
- Safari

### 方式二：安装移动端扩展（可选）

如果你在移动端使用 Trancy，可在下载页的 `Mobile Extension` 区域选择对应入口，例如：

- Firefox Android
- iOS Safari

安装后建议先打开任意网页，确认扩展已启用。

## 前置条件

- 已完成 Trancy 安装并可以正常打开设置页
- 已拥有可用的 AllTokens API Key
- 网络可以访问 `https://alltoken.co`

## 配置步骤

### 步骤 1：打开 Trancy 设置

进入 Trancy 的 `AI` 或 `模型` 相关设置页面。

![Trancy 设置页](/images/usercases/trancy/step-1-settings.png)

### 步骤 2：添加自定义引擎

在引擎管理页点击 **添加自定义引擎**，打开右侧配置面板。

![Trancy 添加自定义引擎](/images/usercases/trancy/step-2-custom-engine.png)

### 步骤 3：填写连接信息

按界面字段逐项填写：

- 选择服务商：`自定义引擎`
- 模型名称：填写你要使用的模型，例如 `gpt-5.4`、`zai-org/GLM-4.5`、`deepseek-ai/DeepSeek-V3`
- 引擎名称：可自定义，例如 `AllTokens`
- API Key：`YOUR_API_KEY`
- API 接口地址：`https://alltoken.co/v1/chat/completions`

### 步骤 4：保存并测试

点击 **添加** 保存配置，然后对任意网页段落进行翻译测试。

如果可以正常返回结果，说明配置成功。

## 地址填写说明

- 当 Trancy 字段是 `API 接口地址` 时，请填写完整地址：`https://alltoken.co/v1/chat/completions`
- 当客户端字段是 `Base URL` 时，请填写：`https://alltoken.co/v1`

## 使用建议

- 先使用基础模型验证链路，再切换到目标模型
- 建议为 Trancy 单独创建 API Key，便于统计与隔离
- 如果有延迟要求，优先选择响应更快的模型

## 常见问题

### 配置后无法请求

请先确认你填写的字段类型：

- `API 接口地址`：`https://alltoken.co/v1/chat/completions`
- `Base URL`：`https://alltoken.co/v1`

### 返回认证失败

请确认 API Key 为 AllTokens 控制台生成的有效令牌，并检查是否包含多余空格。

### 模型不可用

请确认模型名称填写正确，且当前令牌已开通该模型权限。
