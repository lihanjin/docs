## 概述

DB-GPT 支持配置外部模型服务。

通过 Passto AllTokens 作为统一网关，你可以在 DB-GPT 中集中控制模型访问、额度和路由。

## 配置步骤

### 步骤 1：进入模型配置页

打开 DB-GPT 的模型服务或 LLM 配置页面。

### 步骤 2：选择兼容接入方式

如果有 OpenAI 兼容、自定义兼容接口或 HTTP API 配置项，优先使用该方式。

### 步骤 3：填写参数

- API Key：`YOUR_API_KEY`
- Base URL：`https://alltoken.aatest.online/v1`
- Model：例如 `glm-4`

### 步骤 4：执行测试

保存配置后，运行一次简单问答或数据库分析任务，确认模型返回正常。

## 使用建议

- 建议为 DB-GPT 使用独立 API Key
- 如需绑定特定模型能力，可在 Passto AllTokens 侧限制模型范围
