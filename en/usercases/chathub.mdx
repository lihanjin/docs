## Quick setup

Open the [ChatHub browser extension docs](https://doc.chathub.gg/overview/chrome-extension) and install the extension.

Custom bots are only available in the browser extension version.

In ChatHub, choose an OpenAI-compatible or custom provider option and fill in:

- API Key: `YOUR_API_KEY`
- Base URL: `https://alltoken.co/v1`
- Model: for example `glm-4`

Then send a test message to verify the connection.
