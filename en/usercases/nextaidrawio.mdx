## Overview

[Next AI Draw.io](https://github.com/sing1ee/next-ai-drawio) is an AI diagram tool that can create and edit draw.io diagrams from natural language.

With AllToken, you can use a unified API key and model gateway for flowcharts, architecture diagrams, UML diagrams, and mind maps.

## Quick setup

In Next AI Draw.io model settings, configure:

- API Key: `YOUR_API_KEY`
- Base URL: `https://alltoken.co/v1`
- Model: `glm-4` (or any model enabled in your AllToken account)

Save and run the model test before generating diagrams.
